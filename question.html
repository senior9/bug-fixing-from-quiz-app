Difference between forEach,map,filter,find?

ForEach

1. ForEach is a great for simple looping as it's doesn't return any values.

2. ForEach will skip empty array slots.

Map

1. Map create a new collection with changed values with your intial collection.

2. Map will always return something or it values will be undefined.

Filter

1. Filter can be used to return specfic values from source array.

2. Filter will always return something or it values will be undefined.

Find

1. Find search your array and return the first matching element from source array.

2. Find returns any value type.
